# WorkDaySchedulerHW-5
* A simple app for scheduling your work day.


# Features
* HTML
* CSS
* Bootstrap
* jQuery
* Moment.js

# Function
* User is allowed to enter an appointment and save it to local storage.  Data will save and be presented upon refreshing.
